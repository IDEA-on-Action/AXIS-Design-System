# /ax:kpi-digest Command

주간 KPI 요약 리포트를 생성합니다.

## 사용법

```
/ax:kpi-digest [--period <week|month>] [--play <play_id>] [--notify]
```

## 인자

| 인자 | 필수 | 설명 | 예시 |
|------|------|------|------|
| `--period` | | 집계 기간 | week (기본), month |
| `--play` | | 특정 Play만 | EXT_Desk_D01 |
| `--notify` | | Teams/Slack 알림 | |

## 실행 워크플로

**WF-05 KPI Digest**

```
1. Postgres/Confluence DB에서 데이터 조회
2. Play별 전환율/리드타임 계산
3. 목표 대비 페이싱 분석
4. 지연/병목 탐지
5. 리포트 생성
6. (선택) Teams/Confluence 공유
```

## 에러 처리

| 에러 | 메시지 | 해결 방법 |
|------|--------|----------|
| 데이터 없음 | "해당 기간 데이터가 없습니다" | 기간 확인 |
| DB 연결 실패 | "데이터베이스 연결 실패" | 재시도 |

## 관련 커맨드

- `/ax:play-status <play_id>` - 특정 Play 상세 현황
- `/ax:pipeline` - 전체 파이프라인 현황
