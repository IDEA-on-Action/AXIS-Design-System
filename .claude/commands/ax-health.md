# ax-health

프로젝트 상태를 종합적으로 점검합니다.

## 점검 항목

아래 항목들을 순서대로 확인하고 결과를 보고해주세요:

### 1. 의존성 상태
```bash
pnpm install --frozen-lockfile
```
- 락파일과 일치하는지 확인
- 누락된 의존성 보고

### 2. 타입 체크
```bash
pnpm type-check
```
- TypeScript 오류 목록 출력
- 심각도별 분류 (error/warning)

### 3. 린트 검사
```bash
pnpm lint
```
- ESLint 오류/경고 보고
- 자동 수정 가능 항목 표시

### 4. 빌드 테스트
```bash
pnpm build
```
- 각 패키지별 빌드 성공 여부
- 실패 시 오류 원인 분석

### 5. Git 상태
```bash
git status
```
- 커밋되지 않은 변경사항 목록
- 스테이징된 파일 확인

```bash

```
