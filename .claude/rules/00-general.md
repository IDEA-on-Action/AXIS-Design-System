# 일반 규칙 (General Rules)

## 언어 원칙

- **모든 출력은 한글로 작성**: 코드 주석, 커밋 메시지, 문서, 대화 응답
- **예외**: 코드 변수명, 함수명, 기술 용어는 영문 유지

## 날짜/시간 원칙

- **기준 시간대**: KST (Korea Standard Time, UTC+9)
- **날짜 표기**: YYYY-MM-DD 형식

## 작업 실행 원칙

- **병렬 작업 우선**: 독립적인 작업은 항상 병렬로 진행
- **효율성 극대화**: 의존성 없는 도구 호출은 동시에 실행
- **최소 변경 원칙**: 요청된 작업만 수행, 불필요한 확장 금지

## 문서 작업 원칙

- 요청된 작업만 수행 (nothing more, nothing less)
- 파일 생성보다 기존 파일 수정 우선
- 문서 파일(*.md)은 명시적 요청 시에만 생성
