# @axis-ds/cli

AXIS Design System의 CLI 도구입니다. shadcn 호환 방식으로 컴포넌트를 프로젝트에 설치합니다.

## 설치

```bash
npm install -g @axis-ds/cli
# or
npx @axis-ds/cli
```

## 사용법

```bash
# 프로젝트 초기화
axis init

# 컴포넌트 추가
axis add button

# 설치된 컴포넌트 목록
axis list
```

## 문서

자세한 사용법은 [AXIS Design System 문서](https://axis.minu.best)를 참고하세요.
