# PRD — 외부 DS 연합형 확장

> 작성일: 2026-01-31 | WI: WI-0010 | GitHub: #54 | 상태: Draft

---

## 1. 개요

### 1.1 배경
- AXIS DS 단독으로는 모든 UI 요구를 충족하기 어려움
- shadcn, monet 등 외부 디자인 시스템과의 연합 필요

### 1.2 목표
- 외부 DS 컴포넌트를 링크 형태로 통합
- 커뮤니티 기여 가이드 마련
- 라이선스 게이트 자동화

### 1.3 범위
- 포함: shadcn 블록 링크, monet 리소스 연동, 기여 가이드
- 제외: 외부 컴포넌트 직접 fork/복제

---

## 2. 요구사항

### 2.1 기능 요구사항
- FR1: shadcn 블록 링크 템플릿
- FR2: monet 리소스 link-only 연동
- FR3: 커뮤니티 기여 가이드 작성
- FR4: 라이선스 게이트 자동화

### 2.2 비기능 요구사항
- 법적: 라이선스 호환성 검증
- 유지보수: 외부 의존성 최소화

---

## 3. 수용 기준 (Acceptance Criteria)

- AC1: shadcn 블록 연동 가능
- AC2: monet 리소스 링크 동작
- AC3: 기여 가이드 문서 완성
- AC4: 라이선스 검증 자동화 동작
