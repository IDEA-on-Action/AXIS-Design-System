# 2. 프로젝트 개요

---

## 배경 및 문제점

### As-Is: 기존 BD 프로세스의 한계

```
❌ 정보 수집 분산
   - 세미나, 미팅, VOC 등 다양한 채널
   - 담당자별 개인 노트에 기록
   - 공유 및 추적 어려움

❌ 평가 기준 부재
   - 개인 경험에 의존한 판단
   - 기회 우선순위 불명확
   - 리소스 배분 비효율

❌ 긴 의사결정 사이클
   - 수동 문서 작성 (Brief 3~5일)
   - 이메일 기반 승인 (지연 빈번)
   - 전체 리드타임 5주 이상
```
