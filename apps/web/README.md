# AXIS Design System - Web Application

Next.js 15 기반 문서 사이트

## 주요 페이지

- `/` - 메인 페이지
- `/components` - Core UI 컴포넌트 목록
- `/agentic` - Agentic UI 컴포넌트 목록
- `/library` - 컴포넌트 라이브러리
- `/docs` - 설치 가이드

## 개발

```bash
# 개발 서버 실행
pnpm dev

# 빌드
pnpm build
```

## 접속

- **개발 서버**: http://localhost:3100
